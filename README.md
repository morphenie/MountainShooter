Spooky Paws is a 2D arcade game inspired in Mountain Shooter, developed with Pygame. The game has 2 levels and each level end with a timeout event.


Control Keys: 
UP, DOWN, LEFT, RIGHT: MOVE
SPACE: SHOOT
ENTER: SELECT
